# libfuse

仓库包含第三方开源软件libfuse，libfuse是一个用户空间的文件系统框架。


## 目录结构

```
doc/        文档
example/    样例代码
include/    头文件
lib/        libfuse动态库源代码
test/       测试用例代码
util/       fusermount二进制工具
README      README说明
```
